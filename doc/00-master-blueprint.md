# Project Overview

LockUp is a discipline-building platform designed primarily for university students.

The goal is not productivity.

The goal is not task management.

The goal is not planning.

The goal is helping users build discipline through consistent follow-through on commitments.

LockUp helps users:

* Create commitments
* Protect focus
* Block distractions
* Build streaks
* Earn progression
* Visualize growth
* Become more disciplined over time

---

# Core Product Principle

Discipline beats motivation.

Whenever uncertainty exists:

Ask:

"Does this strengthen discipline?"

If yes:

Proceed.

If no:

Reconsider.

## Lock Levels

### Level 1

Can exit.

No penalty.

---

### Level 2

Can exit.

Lose XP.

Lose streak protection.

---

### Level 3

Strict Mode.

Cannot exit.

Timer must complete.

Emergency override available.
